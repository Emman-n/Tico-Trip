Acerca de:

Una de las áreas más hermosas de Costa Rica es el Parque Nacional Volcán Tenorio, pero si no conoce los alrededores, puede perderse las vistas clave y la información que conviene conocer. Este tour por el río Celeste desde San José le permite explorar el área todo el día con un guía que está muy familiarizado con el parque y le muestra el camino. Seguirás el río para ver el volcán Tenorio y más.
